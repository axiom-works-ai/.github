### Philosophy

We believe the best open-source projects are **comprehensive, not minimal**. Every MCP server we publish aims to cover the full API surface — not just transcripts, not just search, but *everything* the platform supports.

When APIs have gaps, we [document them](https://github.com/axiom-works-ai/axiomworks-youtube-mcp/blob/main/API_GAPS.md) and work with platforms to improve coverage.
